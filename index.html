Avec mon groupe scolaire composé de Louis Lamouche et Valentio Pinto et moi même nous sommes actuellement en train de travailler sur un site d'auto-école afin de le présenter le jour de notre épreuve de BTS.
Ce projet à pour but de réaliser la vitrine d'une auto école ainsi qu'une interface utilistateur pour ceux ayant un forfait que l'auto-école.
Grâce a cette interface l'utilistateur pourra consulter son forfait, ses heures effectués, son agenda avec ses prochaines heures de conduites, la possibilité de demander ou annuler des heures et avoir accès à un mini code de la route afin de s'entrainer.
Le site disposera aussi d'une interface moniteur ou il pourra accépter ou non les demandes de ses élèves ainsi qu'accéder à son agenda.
Pour finir il y aura une interface Admin permettant d'insérer ou supprimer un élève et un moniteur

GitHub lance son outil Copilot pour aider les développeurs

Github Copilot est une intelligence artificielle créee par GitHub et OpenAI dans le but d'aider les développeurs.
Elle est capable de générer du code à partir d'un commentaire mais aussi de faire une prédiction de ce que vous voullez potentiellement écrire grace au modèle de Langages GPT3 développer par OpenAI.
Cette AI permet donc la prédiction dans une douzaine de Langages et s'installe sur certains IDE dont Visual Studio Code.
GitHub Copilot est disponible par abonnement mensuel (9.99$) ou annuel (99$) mais dispose aussi d'une offre gratuite pour les étudiants Présenter le 29 juin 2021, GitHub Copilot est maintenant disponible pour les développeurs depuis le 20 juillet 2021.
Afin d'apprendre à coder Github Copilot s'est entrainé sur plus de 1 milliard de lignes de code à travers tous les repository github disponible.
Afin de valider la proposition de github il suffit de cliquer sur la touche TAB et de laisser github copilot faire son travail.
Attention : Github Copilot est encore en phase de test et peut donc proposer des erreurs de syntaxe ou des erreurs de logique
